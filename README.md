## Usage

2. yum -y install zip unzip
3. unzip master.zip
4. cd server_setup-master

5. Edit parameters section of "server_init.sh" 
6. Edit parameters section of "make_vhosts.sh"

8. ./server_init.sh
9. ./make_vhosts.sh
10. ./selinux_off.sh
